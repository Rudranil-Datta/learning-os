Success Criteria (V1 Completion)
User can send a message in main chat and receive an AI answer.

After each answer, the UI shows suggested knowledge nodes (pills) based on the answer.

User can confirm a suggestion → node is saved with unique title constraint enforced.

User can manually create nodes (title + description) via form.

User can link two nodes (parent, child, related).

Knowledge tree displays parent‑child hierarchy.

Clicking a node in the tree opens the contextual AI side panel.

Side panel remembers conversation history per node (messages persist across page reloads).

Side panel answers incorporate the node’s explanation and related concepts.

No PDF ingestion, no vector search, no multi‑user auth – intentionally deferred.
